# Shop Management System
Shop Management System is a web application built with PHP &amp; MySQL to manage inventory, pricing, and profit tracking. It supports Customer and Seller roles, real-time stock updates, automatic profit calculation, bilingual support (English/Sinhala), and a mobile-responsive design.
